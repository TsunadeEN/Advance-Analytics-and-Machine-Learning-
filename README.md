# Advance-Analytics-and-Machine-Learning-
Understanding Factors Influencing Scientific Citations 
Citations serve as a proxy for academic impact. This study explores how factors like publication year,
network connectivity, grants, and patents influence the number of citations an article receives.
Weapplyrigorousexploratorydataanalysis(EDA),generalizedadditivemodeling(GAM),tree-basedmodels
(Decision Trees, Random Forests, Gradient Boosting), and unsupervised learning (PCA, k-means clustering,
t-SNE).
